# Example for the LIS3DH

Example to read 3-axis acceleration data from the LIS3DH sensor.

## Description

This example will continuously read and print the acceleration values for X, Y, and Z axes. The sensor is configured with:
- Output data rate: 100 Hz
- Full scale range: Â±2g
- I2C interface

The acceleration values are printed every 100ms to the console.
