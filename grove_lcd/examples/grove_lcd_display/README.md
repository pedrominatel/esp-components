# Grove LCD Display Example

This example demonstrates basic text display and backlight control features of the Grove LCD RGB Backlight v4.0.

## Description

This example will:
1. Initialize the Grove LCD display
2. Display text on both rows
3. Demonstrate RGB backlight color changes (red, green, blue, white)
4. Show cursor visibility and blinking control
5. Display a simple counter animation

The display shows various text and color combinations to showcase all basic features.
